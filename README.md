# Database Bank
* It is a small website for the Data Management of the Army Personal Details

### Requirements
* Ruby 3.2.2
* Rails 7
* Mysql

### Setup Guide
* bundle install
* bin/rake db:setup
* bin/rake db:migrate
* bin/rake db:seed
* bin/rails s
